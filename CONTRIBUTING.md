# Contributing to AI Company Platform

We love your input! We want to make contributing to AI Company Platform as easy and transparent as possible.

## Development Process

1. Fork the repo and create your branch from `main`
2. If you've added code that should be tested, add some tests
3. If you've changed APIs, update the documentation
4. Ensure the test suite passes
5. Make sure your code lints
6. Issue that pull request!

## Code Style

- Use consistent formatting
- Write meaningful commit messages
- Comment your code when necessary
- Follow the existing patterns in the codebase

## Pull Request Process

1. Update the README.md with details of changes if applicable
2. Update the API_DESIGN.md if you changed the API
3. The PR will be merged once you have the sign-off of at least one maintainer

## Any Questions?

Feel free to open an issue for any questions!
